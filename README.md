# Spotify Clone

A frontend Spotify-inspired web interface built using **HTML5** and **CSS3**. This project recreates the visual design of Spotify with a clean layout, responsive sections, and modern music player styling.

## Tech Stack

* HTML5
* CSS3

## Features

* Spotify-inspired homepage UI
* Sidebar navigation menu
* Music playlist cards
* Album and artist sections
* Bottom music player interface
* Responsive layout design
* Hover effects and transitions

## What Makes This Project Special

This project focuses on recreating a real-world music streaming interface while improving frontend design skills.

It demonstrates strong understanding of layout structuring, Flexbox, responsive design, and UI styling techniques. The project also helped strengthen skills in building pixel-inspired clones of production-level interfaces.

## Learning Outcomes

* HTML semantic structure
* CSS Flexbox
* Responsive design
* UI cloning
* Layout design principles
* Styling and hover interactions

## Limitations

This is a frontend-only project.

Not included:

* JavaScript functionality
* Audio playback
* Backend integration
* Authentication system
